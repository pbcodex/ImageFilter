# ImageFilter
Une DLL et un include pour le langage PureBasic pour le traitement d'images crée par G-Rom.

- GreyScale
- Luminosity
- Contrast
- Gamma
- Seuil 
- Inverse (Négatif d'une image)
- Normal Map
- Filtre Sharpen
- Filtre Motion Blur
- Filtre Emboss
- Filtre Find Edges 
- Filtre Blur
- Filtre Gaussian
